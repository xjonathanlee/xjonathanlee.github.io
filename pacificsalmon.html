Pacific Salmon Foundation

Pacific Salmon Foundation is a scientific driven research and governing body that collectively looks at conservation, restoration and the historical documentation of the Salmonid population in British Columbia. I was brought in to revisualize key components in their new 'Pacific Salmon Explorer' that helps people make informed goverment level choices on infrastructure that could have impact on Pacific Salmon.

  • Roles:Visual Design, User Interface Design, User Heuristic Designer, Mapbox Coding
  • Tools: Adobe Illustrator, Adobe Photoshop, Mapbox, Figma
  • Year: 2024
  • Context: Client Internship (6 monthes)
  • Deliverables:

Problem

How can we design a better user experience for goverments to make informed choice on projects that could impact future salmon stocks?

Design Process

Designing for the User

I proposed multiple solutions that looked at new features to color reworks that better streamlined how users of tool interacted with the PSF. In the initial discovery phase I was brought on to initially look at color and patterns, but as the tool grew in size and scope I was asked to reinterpret data, form new user patterns and implement general design changes to sections that linked to other sites.

Some of my contributions can be found in the list below:

  1. A new feature that enables users to save charts they deem important
  2. A reworked color and pattern system that better looks interactions between colors and patterns instead of stand-alone
  3. An introduction of new user patterns that scaffold interactions
  4. A complete redesign of how data is interpreted in client side graphs
  5. A overhauled tooltip system
  6. A ground-up rework of the Regional Overviews section that links with an external site.

Interactions + Redesign features

By allowing users to pace themselves, one can look at the content as a way to inform users without overwhelming them. Given a high level overhead combined with a reflection section that drives home intent across different levels of media (mixing article and scroll interactions), we can ensure users adopt and process the message we are telling them instead of mindlessly scrolling.
