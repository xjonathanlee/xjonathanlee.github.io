Musical DNA

Samurai Champloo expanded my knowledge in music and introduced me to Seba Jun at 13. When he passed in 2008 I was intimately familiar with his music that fused both jazz and hiphop. This project was a response to his albums he produced both solo and with his collective.

  • Roles:Art Direction, Research Creation
  • Tools: Adobe Illustrator, Interaction Design, Data Synthesis
  • Status: 2 Weeks
  • Context: Personal Project
  • Deliverables: Poster, Dataset

Problem

Excersing our skills in data design how might we abstract Seba Jun's discography?

Project Brief

From the start I knew that I wanted to create a long form poster that would feature a way to catalogue DNA from across Nujabes entire discography, something like how you would read a DNA CRSPR slide only more visual. This process of creating a reasonable yet easy to read visual took a long, but fun process of iteration.

Added Functionality

With the initial poster in play, I wanted to see if I could push functionality to allow for a better user experience digitally. I chose to do this by adding a holistic zoom feature that allowed users to view things more closely.
